# LinkedIn Hooks

Write compelling LinkedIn post hooks that stop the scroll and drive engagement.

**Author:** HonePrompt Team
**Estimated Cost:** $1-3 (25 iterations)
**Tags:** linkedin, social-media, hooks, copywriting

## What This Template Does

Optimizes a LinkedIn hook-writing prompt. The prompt takes a post topic as input and generates attention-grabbing opening lines (1-2 sentences) designed to maximize "see more" clicks.

## Scoring Criteria

- **Hook strength (40%)** — Does it stop the scroll? Create genuine curiosity?
- **Specificity (25%)** — Does it use concrete details, not vague claims?
- **Brevity (20%)** — Is it under 20 words? Every word essential?
- **Authenticity (15%)** — Does it feel real, not clickbait?
