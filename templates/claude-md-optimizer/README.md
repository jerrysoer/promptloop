## What This Template Does

Optimizes a prompt that generates CLAUDE.md instruction files for AI coding assistants. Given a project description and tech stack, the optimized prompt produces focused, actionable context files.

## Scoring Criteria

- **Actionability (30%)** — Contains concrete commands and patterns
- **Specificity (25%)** — Tailored to the specific project
- **Completeness (25%)** — Covers commands, architecture, conventions, gotchas
- **Conciseness (20%)** — Under 50 lines, no fluff
